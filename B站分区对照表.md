## 四、每个一级分区下的二级分区

### 动画 → `/anime`
- MAD·AMV、MMD·3D、短片·手书·配音、手办·模玩、特摄、动漫杂谈、综合

### 番剧 → `/anime`（合并）
- 连载动画、完结动画、资讯、官方延伸、新番时间表

### 国创 → `/guochuang`（新增）
- 国产动画、布袋戏、动态漫·广播剧、资讯

### 音乐 → `/music`
- 原创音乐、翻唱、VOCALOID·UTAU、电音、演奏、MV、音乐现场、说唱

### 舞蹈 → `/dance`（新增）
- 宅舞、街舞、中国舞、舞蹈教程、舞蹈综合、舞蹈翻跳

### 游戏 → `/game`
- 单机游戏、电子竞技、手机游戏、网络游戏、桌游棋牌、GMV、音游、Mugen

### 知识 → `/knowledge`（新增）
- 科学科普、社科·法律·心理、人文历史、财经商业、校园学习、职业职场、设计·创意

### 科技 → `/tech`（新增）
- 数码、软件应用、计算机技术、科工机械、极客DIY

### 运动 → `/sports`（新增）
- 篮球、足球、健身、竞技体育、运动文化、运动综合

### 汽车 → `/car`（新增）
- 赛车、购车攻略、汽车生活、汽车知识、摩托车

### 生活 → `/life`（新增）
- 搞笑、日常、亲子、家居房产、手工、绘画、出行、三农、生活记录

### 美食 → `/food`（新增）
- 美食制作、美食侦探、美食测评、田园美食、美食记录、街头美食

### 动物圈 → `/animal`（新增）
- 喵星人、汪星人、大熊猫、小宠异宠、野生动物

### 鬼畜 → `/kichiku`（新增）
- 鬼畜调教、音MAD、人力VOCALOID、鬼畜剧场

### 时尚 → `/fashion`（新增）
- 美妆护肤、穿搭、风尚潮流、时尚购物、发型

### 娱乐 → `/entertainment`（新增）
- 综艺、娱乐杂谈、粉丝创作、明星综合

### 影视 → `/movie`（合并电影+电视剧+影视）
- 影视杂谈、影视剪辑、短片、预告·资讯、影视评论

### 纪录片 → `/documentary`（新增）
- 人文·历史、科学·探索·自然、军事、社会·美食·旅行、纪实

### 电影 → `/movie`（合并）
- 华语电影、欧美电影、日本电影、韩国电影

### 电视剧 → `/movie`（合并）
- 国产剧、海外剧、港台剧、韩剧、日剧
