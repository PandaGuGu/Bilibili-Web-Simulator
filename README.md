# 🚀 快速开始

## 安装依赖
```bash
npm install
```

## 启动开发服务器

### 普通用户端口（5173）
```bash
npm run dev:user
```
访问地址：http://localhost:5173/

### 管理员端口（5174）
```bash
npm run dev:admin
```
访问地址：http://localhost:5174/

## 构建生产版本
```bash
npm run build
```
